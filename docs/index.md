# Azure SQL Database container

> **Version:** June 18, 2026
> **Status:** Private Preview
> **Generated by:** SQL Developer Experiences Team @ Microsoft

The Azure SQL Database container is a developer-focused, OCI-compliant container image of the SQL Database engine. Build, test, and ship modern applications against Azure SQL Database without an Azure subscription, without a shared cloud instance, and without changing connection strings between the inner loop and production.
